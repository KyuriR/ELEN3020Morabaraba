# Project Name- Morabaraba2
After extracting the project files, all game scripts are located in the Assets/Scripts directory.

## Description
This project is a Unity-based Morabaraba prototype developed as part of the ELEN3020A software engineering project. 
The game includes online multiplayer support through 
Photon PUN 2, AI opponents, customisable themes, leaderboard functionality using LootLocker, and beginner move-hint systems.

## Features
- Multiplayer support
- Turn-based gameplay
- Photon PUN 2 networking

## How to Run
1. Open in Unity
2. Load the main scene
3. Press Play
